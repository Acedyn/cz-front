# Image Region 2D

The image region is meant to be used as a child in the ``elements`` slot of the
``Scene2D`` component which aims to simplify the creation of scenes.

## Slots

#### Hover

Insert any element that will be displayed on hover, to add details about the section

## Hint

The ``config`` props should be generated by the ``Scene2D`` component. The idea is that
a global global configuration can be set at the scene level, and then this config is
passed to each elements. This is to avoid having to set the same props on every elements
of a scene
